# Protocolo preliminar de auditoría

## Actividad 1

Diagnóstico regional de aptitud de fuentes y vacíos de información.

## Reglas iniciales

1. No modificar datos originales.
2. No limpiar datos antes de auditar.
3. No descartar registros sin documentación.
4. Separar datos crudos, intermedios y procesados.
5. Registrar toda decisión metodológica.
6. No subir datos geoespaciales pesados a GitHub.
